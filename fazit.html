Fazit

Besonders eines sollte klar geworden sein: Eine Kugel und ein Kreis sind fast ein- und dasgleiche. Beide beschreiben eine Menge von Punkten, die einen vorgegebenen Abstand zu einem Ausgangspunkt haben. Der einzige Unterschied besteht in den unterschiedlichen Dimensionen. Dass sich Kreis und Kugel sehr ähneln, merkt man auch daran, dass die Gleichungen sehr ähneln oder sogar gleich sind. Die einzige wichtige Formel, die man sich merken sollte, ist: $$ (\vec{x}-\vec{m})\cdot (\square -\vec{m})=r^2 $$

  • Setzt man für $\square$ den Vektor $\vec{x}$ ein, so erhält man die Kreis-/Kugelgleichung.
  • Setzt man für $\square$ den Vektor $\vec{b}$ ein (B liegt auf Kreis/Kugel), so erhält man die Gleichung für eine Tangente/Tangentialebene.
  • Setzt man für $\square$ den Vektor $\vec{p}$ ein (P=Pol liegt außerhalb des Kreises/der Kugel), so erhält man eine Gleichung für die Polare.
Viele Aufgaben kann man einfach lösen, indem man sich die geometrischen Objekte vorstellt. Programme wie GeoGebra (damit wurde auch diese Website erstellt) können dabei sehr hilfreich sein.

Natürlich wurde das Thema nicht vollständig behandelt. Es gibt noch viele weitere interessante Punkte. Zum Beispiel:

  • die Koordinatengleichungen von Kreisen, Kugeln, Tangenten, Tangentialebenen, Polaren, ...
  • die Parametergleichungen von Kreisen, Kugeln, Tangenten, Tangentialebenen, Polaren, ...
  • die Untersuchung von höheren Dimensionen
  • die Untersuchung von anderen Kurven z. B. Cycloide in 2d, oder Kegel in 3d
  • ...
